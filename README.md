# ESI Algiers Switch Configuration Tool
This is a web-based tool for the network department at ESI Algiers that allows administrators to manage the location and configuration of switches across the campus. The tool is built using the MERN stack and integrates with the department's existing database to fetch and update switch data.

# Features
View a list of all switches on the network, including their location, model, and status
Add new switches to the database, including their location and configuration
Edit existing switch information, including VLAN configurations and network settings
Search for switches by name or location
Authentication and role-based access control to limit access to sensitive switch data
Export switch data to CSV or JSON formats for backup or reporting purposes


# Technologies
MongoDB for database management
Express for API development
React for the front-end user interface
Node.js for server-side development
Material UI for component design

# Getting Started
To get started with the tool, you'll need to have Node.js and MongoDB installed on your machine. Clone the repository and run npm install to install dependencies. You'll also need to set up the database connection string to connect to your department's MongoDB instance.
